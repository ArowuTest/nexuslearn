Teaching Journey

warm_up

letter-path-glow

Follow the glowing path with your finger.

Prepare movement path before assessment.

Audio: Follow the path slowly. Start at the glowing dot.

Visual: Large letter path with animated start dot.

concept_launch

trail-marker-move

The letter trail shows where to start and which way to go.

Introduce handwriting as movement, not only shape.

Audio: Letters have a start, a path and a finish.

Visual: A trail marker moves around a lowercase letter.

worked_example

letter-c-trace

For c, start near the top and curve around.

Model a simple curved letter path.

Audio: Start near the top. Curve around like a small moon.

Visual: Letter c drawn with a soft trail.

guided_practice

trace-path-feedback

Trace the letter path, then try again with less help.

Practise tracing with fading support.

Audio: Stay close to the trail. It is okay to try again.

Visual: Trace canvas with wide tolerance and replay button.

checkpoint

start-dot-choice

Tap where this letter should start.

Check start point without requiring motor tracing.

Audio: Find the start point before you trace.

Visual: Three possible start dots around a letter.

teach_back

companion-letter-repair

Teach Pip why the start point matters.

Reinforce movement path through explanation.

Audio: I started at the bottom and got muddled. Can you show me the start?

Visual: The companion compares two letter paths.

Adaptive Support

low_sensory

Use soft trails and no flashing correction marks.

reduced_motion

Use static arrows and step markers.

audio_first

Narrate each movement with replay.

reading_support

Use letters, sounds and icons instead of written instructions.

attention_support

Practise one letter movement at a time.

processing_support

Allow unlimited replay and no timer.

confidence_support

Separate motor effort from learning judgement.

eal_support

Preteach start, finish, curve, line, round and trace with gestures.
