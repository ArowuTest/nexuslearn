Teaching Journey

warm_up

seed-tap-count

Tap each seed once as we count together.

Check one-to-one correspondence before abstract number sequences.

Audio: Touch one seed for each number. One number, one seed.

Visual: Seeds glow one at a time in a small garden row.

concept_launch

number-path-reveal

The garden path can start anywhere. What comes after 27?

Introduce counting on from a given number rather than restarting.

Audio: We do not need to start at one. We can start at twenty-seven and count on.

Visual: A numbered path begins at 27 with the next stones hidden.

worked_example

decade-bloom

After 29 comes 30. Watch the new ten-frame fill.

Model the decade transition that often causes errors.

Audio: Twenty-nine has two tens and nine ones. One more makes thirty.

Visual: Nine loose seeds join a full group to become three tens.

guided_practice

stone-snap-sequence

Drag the missing numbers onto the path.

Practise forward and backward sequences with visual support.

Audio: Look at the number before and after. Which number fits?

Visual: A number path with draggable stone labels.

checkpoint

backward-path-dim

Count backwards from 43 to 39.

Check counting backwards across a decade boundary.

Audio: Each step backwards takes away one.

Visual: Four stepping stones dim from 43 down to 39.

teach_back

companion-start-marker

Teach Pip how to count on from 58 without starting at 1.

Use explanation to confirm flexible starting points.

Audio: Can you show me where to start and what number comes next?

Visual: The companion waits beside a partly hidden number path.

Adaptive Support

low_sensory

Use gentle garden colours, soft glows and no burst effects.

reduced_motion

Reveal number stones instantly with static focus outlines.

audio_first

Read all numbers and prompts aloud with replay and slower-count controls.

reading_support

Use numerals, spoken prompts and picture cues rather than long written instructions.

attention_support

Keep each interaction to one clear action: tap, drag or choose.

processing_support

Allow pauses between count steps and let learners replay the sequence.

confidence_support

Frame mistakes as the path asking for a second look, with no red failure state.

eal_support

Preteach before, after, forwards, backwards, more and less with animated examples.
