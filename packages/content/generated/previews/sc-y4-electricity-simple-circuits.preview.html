Build a complete loop so the bulb can light.

Introduce circuit completeness as an unbroken path.

current-loop-trace

Watch the path leave the cell, pass through the bulb, and return to the cell.

Model a complete series circuit and return path.

Audio: The path goes out from one terminal and returns to the other terminal.

Adaptive Support

low_sensory

Use steady glows and avoid flashing or alarm-style buzzer animations.

reduced_motion

Use static numbered path segments instead of animated current traces.

audio_first

Narrate component names, circuit tests and switch states with replay.

reading_support

Use component icons and short labels before written explanations.

attention_support

Limit the builder tray to components needed for the current task.

processing_support

Allow step-by-step test mode that highlights one connection at a time.

confidence_support

Use debugging language: the circuit has a clue, not the learner has failed.

eal_support

Preteach cell, wire, bulb, switch, open, closed, complete and gap.
