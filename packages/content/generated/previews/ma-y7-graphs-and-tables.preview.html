Teaching Journey

warm_up

table-bar-link-scan

Match each table value to the right bar on the graph.

Check that learners connect categories, rows and values before interpreting patterns.

Audio: Read the category first, then follow across the row to its value.

Visual: A two-column table and a bar chart light up matching category-value pairs.

concept_launch

axis-builder-assemble

Build the axes before plotting: title, labels, units and equal scale steps.

Make graph construction a set of deliberate design decisions rather than decoration.

Audio: A trustworthy graph needs labelled axes, units and a scale with equal steps.

Visual: An empty graph frame gains axis labels, unit badges and evenly spaced tick marks.

worked_example

scale-tick-count

The vertical scale counts in 5s. What value does this point show?

Model careful scale reading before answering from graph position.

Audio: First inspect the scale. Each tick is five, so halfway between ten and fifteen is twelve point five if the graph allows decimals.

Visual: Tick intervals are counted aloud before a reading ruler aligns with the data point.

guided_practice

scale-repair-snap

Repair this misleading graph by choosing an even scale and adding missing labels.

Practise diagnosing uneven intervals, missing units and compressed or exaggerated patterns.

Audio: A repaired graph uses equal gaps for equal number changes and tells the reader what each axis measures.

Visual: Uneven ticks straighten into equal intervals while warning markers turn into label prompts.

checkpoint

graph-evidence-thread

Which claim is supported by the graph, and which exact values prove it?

Check evidence-based interpretation rather than impressionistic graph reading.

Audio: A graph claim should point to the data values that support it.

Visual: Claim cards connect to highlighted bars, points or table cells.

teach_back

companion-scale-repair

Teach Nova why a graph with uneven scale steps can make a small change look huge.

Consolidate scale fairness and reader trust through explanation.

Audio: I made the gaps different sizes to fit the data. Why might that mislead the reader?

Visual: The companion compares a misleading graph with a repaired graph using the same data.

Question Variants

ma-y7-graphs-and-tables-q-read-scale

graph-reader review difficulty 4

The vertical axis counts in 5s. What value is shown by the point above Wednesday?

The point above Wednesday lines up with 25 on a vertical axis that counts in 5s.

ma-y7-graphs-and-tables-q-build-axis

axis-builder review difficulty 5

Choose the best y-axis label and scale for a bar chart of how many pupils chose each lunch option.

The y-axis should be labelled Number of pupils, and counting from 0 to 20 in equal steps of 2 fits all the values.

ma-y7-graphs-and-tables-q-scale-repair

scale-repair review difficulty 6

A graph uses y-axis ticks at 0, 4, 5, 15 and 20 with equal visual gaps. What repair makes the scale fair?

A fair axis has equal visual gaps for equal numerical steps, so 0, 5, 10, 15, 20 repairs the uneven scale.

ma-y7-graphs-and-tables-q-supported-claim

claim-evidence review difficulty 5

Which claim is supported by the table, and which values prove it?

Coding has 17 attendees, which is greater than Art's 14, Drama's 12 and Chess's 9.

ma-y7-graphs-and-tables-q-line-graph-limit

graph-reader review difficulty 7

A plant was measured once each week. Which statement is safest from the line graph data?

The measured values rise from 8 cm to 18 cm, but the graph does not prove exact daily growth or a future height.

Adaptive Support

low_sensory

Use a calm data-board theme, stable focus outlines and no flashing warning states or rapid graph transitions.

reduced_motion

Replace animated plotting, snapping and trend tracing with static before-after panels and numbered build steps.

audio_first

Read graph title, axis labels, units, scale interval and selected data values in a consistent order, with replay for each bar, point or cell.

reading_support

Offer simple text tables, line focus, glossary popovers, short prompts and symbol-supported labels for axis, scale, category and value.

attention_support

Reveal one decision at a time: read table, inspect axis, choose scale, plot/read value, then make a claim.

processing_support

Allow a graph-reading ruler, tick-count checklist, table focus mode and extra wait time before submitting.

confidence_support

Preserve correct data reading as partial success and route only the failed axis, scale or evidence decision to repair.

eal_support

Preteach axis, scale, interval, value, category, trend, outlier, evidence and compare with visual glossary cards.

switch_access

Provide linear scan order across table cells, axis controls, graph marks and answer cards, with no task requiring drag-only input.

simple_text

Every graph item includes a text-table equivalent and a concise description of axis labels, scale and selected marks.
