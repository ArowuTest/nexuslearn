Teaching Journey

warm_up

enquiry-type-sort

Match each science question to the enquiry type that would answer it best.

Show that not every enquiry is a fair test and that method choice depends on the question.

Audio: Scientists choose the enquiry type that fits the question. Some questions need a fair test; others need observation, classification or pattern seeking.

Visual: Question cards slide to fair test, observation, classification and pattern-seeking trays.

concept_launch

variable-cards-lock

Choose the variable to change, the variable to measure and two variables to keep the same.

Introduce variable control as the heart of fair-test planning.

Audio: For this fair test, change one thing, measure one thing and keep the important other things the same.

Visual: A lab bench with change, measure and keep-same slots.

worked_example

method-plan-build

Plan how to test whether the number of cells affects bulb brightness in a safe circuit model.

Model a safe method with controlled variables, measurement and repeats.

Audio: We change the number of suitable cells, measure brightness on the meter, and keep the bulb, wires and switch the same.

Visual: Circuit investigation plan with safety note, variables, results table and repeat rows.

guided_practice

data-format-bridge

Record the results in the clearest table, then choose a graph if it helps answer the question.

Connect data format to the type of evidence gathered.

Audio: A table keeps results organised. A graph can help reveal a pattern, but only when the graph matches the data.

Visual: Results table linked to bar chart, line graph and scatter graph choices.

checkpoint

evidence-row-highlight

Choose the conclusion that uses the results instead of guessing.

Check claim-evidence reasoning and reject unsupported conclusions.

Audio: A scientific conclusion should point back to the evidence. It should not just repeat what we hoped would happen.

Visual: Three conclusion cards, with evidence rows lighting when a conclusion is supported.

teach_back

companion-method-repair

Teach Nova one improvement that would make the investigation more reliable.

Consolidate evaluation: repeats, accuracy, controls and clearer measurement.

Audio: My result was surprising, so I will just change everything next time. What would be a better scientific improvement?

Visual: Companion compares repeat measurements, changed controls and clearer measuring equipment.

Question Variants

sc-y6-working-scientifically-q-variables

fair-test-plan review difficulty 5

In a safe model testing whether more cells make a bulb brighter, which variable should change?

The number of suitable cells is the variable changed; brightness is measured and other important parts are controlled.

sc-y6-working-scientifically-q-recording

observation-record review difficulty 5

Why is a results table useful before choosing a graph?

A results table helps organise measurements and makes it easier to check patterns, anomalies and graph choices.

sc-y6-working-scientifically-q-conclusion

claim-evidence-explain review difficulty 6

A plant grew 2 cm without fertiliser and 5 cm with fertiliser in a controlled classroom test. Which conclusion uses the evidence carefully?

The careful conclusion uses the results from this test and avoids claiming more than the evidence supports.

sc-y6-working-scientifically-q-graph-choice

graph-table-investigation review difficulty 6

You measured shadow length every hour. Which display would best show how it changed over time?

A line graph can show change over time when the axes are labelled clearly.

sc-y6-working-scientifically-q-reliability

claim-evidence-explain review difficulty 7

Nova has one surprising measurement. What is the best next step?

Repeating the measurement and checking the method helps decide whether the result is reliable or anomalous.

Adaptive Support

low_sensory

Use calm lab benches, no timed pressure and no unsafe or messy experiment imagery.

reduced_motion

Show method steps and data displays as static cards with optional highlights.

audio_first

Narrate each variable slot, table row and conclusion choice with replayable chunks.

reading_support

Use short labels, icons for change/measure/same and simple-text conclusion frames.

attention_support

Work through one enquiry phase at a time: question, variables, method, data, conclusion, improvement.

processing_support

Offer a method checklist and allow learners to freeze the model before recording evidence.

confidence_support

Present anomalies as scientific puzzles rather than failure.

eal_support

Preteach variable, fair, reliable, evidence, prediction and conclusion with sentence frames.
