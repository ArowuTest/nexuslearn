variable-rule-machine

One rule can work for many inputs: output equals three times n plus two.

Introduce a variable as a general number in a reusable rule.

Audio: The letter n can take different number values, but the rule stays the same.

Visual: Several input cards pass through the same 3n + 2 function machine.

worked_example

substitution-step-lock

For c = 5, evaluate 4c + 3 by replacing c with 5.

Model complete substitution and operation order.

Audio: Replace c with five: four times five, then add three, gives twenty-three.

Visual: The 5 tile replaces c before the multiplication and addition stages light in order.

guided_practice

equation-balance-undo

Build both sides of 3n + 2 = 20, then find n without tipping the balance.

Connect inverse operations with equality and symbolic unknowns.

Audio: Keep both sides equal. Undo plus two, then undo times three.

Visual: A balance model removes two from both sides and partitions the remaining eighteen into three equal groups.

checkpoint

solution-pair-tester

Which pairs make a + b = 12 true?

Diagnose whether the learner recognises multiple valid solutions.

Audio: Test each pair by checking whether the two values total twelve.

Visual: Coordinate-like pair cards slot into an equation tester.

teach_back

coach-equality-repair

Teach the coach why 7 + 5 = 9 + 3 is true even though neither side is just an answer.

Secure a relational understanding of the equals sign before Year 7 algebra.

Audio: Explain what the equals sign says about the value on each side.

Visual: The coach points at two expression cards on a level balance.

Question Variants

ma-y6-algebra-readiness-q-formula

function-machine review difficulty 5

A shape pattern has T = 4n + 1 tiles in figure n. How many tiles are in figure 6?

Substituting n = 6 gives T = 4 × 6 + 1 = 24 + 1 = 25.

ma-y6-algebra-readiness-q-equivalence

equation-balance review difficulty 5

Which number makes 18 + 7 = 20 + n true?

The left side equals 25, so 20 + n must also equal 25. Therefore n = 5.

ma-y6-algebra-readiness-q-symbolise

symbol-build review difficulty 4

Sam has p stickers. Jo has 8 more stickers than Sam. Build an expression for Jo's stickers.

Jo has Sam's p stickers plus 8 more, so the expression is p + 8.

ma-y6-algebra-readiness-q-pairs

multiple_choice review difficulty 8

For non-negative whole numbers, which list gives every pair (a, b) with a < b that satisfies a + b = 8?

The four listed pairs total 8 and meet a < b. The pair (4, 4) is excluded because the values are equal.

Adaptive Support

low_sensory

Use a calm workbench, steady highlights and no tipping or alarm effects for incorrect equations.

reduced_motion

Replace moving balance pans and travelling tiles with static before-and-after states.

audio_first

Read symbols as complete mathematical phrases and replay each operation stage separately.

reading_support

Pair algebraic notation with bar models, plain-language rules and uncluttered symbol cards.

attention_support

Reveal one stage at a time: represent, substitute, calculate, then check equality.

processing_support

Allow unlimited working time, an operation-order strip and persistent display of the given variable value.

confidence_support

Normalise unknowns as familiar missing-number problems and credit correct representation separately from arithmetic accuracy.

eal_support

Preteach variable, unknown, formula, substitute, expression and equals with visual examples and sentence frames.
